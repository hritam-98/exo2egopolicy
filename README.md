# Exo2EgoPolicy

**Geometry-Aware Policy Transfer from Exocentric Human Demonstrations**

Hritam Basak, Hadi Tabatabaee, Xin Yang, Shreekant Gayaka, Nan Qiao, Yuyin Sun, Cheng-Hao Kuo, Zhaozheng Yin, Min Sun

*Amazon.com · Stony Brook University*

---

Exo2EgoPolicy is a geometry-aware framework that learns egocentric robot manipulation
policies from monocular exocentric (third-person) human video via viewpoint-aligned
**pose translation**. Instead of synthesizing egocentric pixels, it translates the
hand–object kinematic state on the $SE(3)$ manifold, producing temporally consistent
poses that are co-trained with limited robot teleoperation data.

This repository hosts the [project website](https://hritam-98.github.io/exo2egopolicy/).
